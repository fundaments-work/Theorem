# Lion Reader - Roadmap

## Core Philosophy

**"Read at the speed of thought. Absorb, retain, and master knowledge faster than ever before."**

---

## Features

### Phase 1: Core Reading
- EPUB Support
- MOBI/AZW/AZW3 Support
- FB2 Support
- CBZ/CBR Support
- Library Management
- Book Progress Tracking
- Annotations & Highlights
- Bookmarks
- Table of Contents Navigation
- Search within Books
- Reading Statistics

### Phase 2: Learning Tools
- Dictionary Integration
- Vocabulary Builder
- Spaced Repetition System (SRS)
- Daily Review System

### Phase 3: Content Discovery
- RSS Feed Reader
- Web Clipper (Browser Extension)
- Article Saving & Reading
- Newsletter Subscriptions

### Phase 4: Academic Features
- arXiv Integration
- PubMed Integration
- Academic Paper Support
- Reference Management
- Citation Export

### Phase 5: Integrations
- Obsidian Export
- Public API

### Phase 6: Advanced Features
- Text-to-Speech (TTS)
- Velocity Mode (Speed Reading)
- Sync & Cross-Platform
- Mobile Apps

---

## Technology Stack

- React + TypeScript
- Tauri (Rust backend)
- Tailwind CSS
- Zustand (state management)
- Foliate-js (EPUB/MOBI/CBZ rendering)
